## 🎯 核心方法论

### 好问题的五个基本特征
1. 理解一致性 - 所有受访者理解一致
2. 管理一致性 - 问题能一致呈现
3. 答案类型一致性 - 明确期望的答案形式
4. 可回答性 - 受访者拥有所需信息
5. 愿意回答 - 受访者愿意提供正确答案

### 事实问题设计五步法
1. 界定目标 → 明确信息类型
2. 概念界定 → 确保术语理解一致
3. 记忆策略 → 克服回忆障碍
4. 答案形式 → 能用要求术语回答
5. 社会期望控制 → 减少偏差影响

### 预调查评估三部曲
1. 专题小组讨论 - 探索理解差异
2. 深度个人访谈 - 探查认知过程
3. 田野预调查 - 模拟真实情境

### 效度评估四策略
1. 研究相关性分析
2. 记录核对验证
3. 替代形式比较
4. 再测一致性检验

## 📖 使用方法
