# 问卷设计检查清单

## 一、问题质量检查（五特征检核）

### 1. 理解一致性
- [ ] 所有关键术语是否已明确界定？
- [ ] 是否存在可能产生歧义的词语？
- [ ] 不同受访者是否可能理解不同？
- [ ] 概念范围是否清晰（包含什么/排除什么）？

### 2. 管理一致性
- [ ] 书面形式：受访者能否读懂？
- [ ] 访员访谈：访员能否逐字念出？
- [ ] 电话访谈：问题是否适合口头表达？
- [ ] 自填问卷：格式是否清晰易懂？

### 3. 答案类型一致性
- [ ] 问题是否明确说明期望的答案形式？
- [ ] 受访者是否清楚"什么是充分答案"？
- [ ] 答案单位是否明确（年/月/次/金额）？
- [ ] 对于多答案问题，是否说明了答案数量？

### 4. 可回答性
- [ ] 受访者是否拥有回答所需信息？
- [ ] 记忆任务是否在合理范围内？
- [ ] 时间定位是否可行？
- [ ] 问题难度是否与受访者能力匹配？

### 5. 愿意回答
- [ ] 问题是否会引起社会期望偏差？
- [ ] 敏感问题是否有适当的引入说明？
- [ ] 是否提供了私密性保护？
- [ ] 答案是否会被视为对受访者的评判？

---

## 二、问题措辞检查

### 基本原则
- [ ] 问题是否简洁明了？
- [ ] 是否只问一个问题（无双重问题）？
- [ ] 是否避免了否定式表述？
- [ ] 是否避免了诱导性措辞？
- [ ] 是否避免了专业术语（除非必要且已解释）？
- [ ] 问题长度是否适中（建议不超过20字）？

### 术语界定
- [ ] 所有专业术语是否已定义？
- [ ] 模糊概念是否有示例说明？
- [ ] 时间范围是否明确？
- [ ] 行为标准是否具体？

### 答案选项
- [ ] 选项是否穷尽（覆盖所有可能）？
- [ ] 选项是否互斥（不重叠）？
- [ ] 选项顺序是否平衡（避免顺序效应）？
- [ ] 选项标签是否清晰？
- [ ] 是否提供了"不知道/不适用"选项（如需要）？

---

## 三、问卷结构检查

### 整体组织
- [ ] 问题顺序是否符合逻辑？
- [ ] 是否从简单问题开始？
- [ ] 敏感问题是否放在后面？
- [ ] 相似主题的问题是否分组？
- [ ] 跳答逻辑是否清晰且必要？

### 格式设计
- [ ] 访员指令是否与问题明确区分？
- [ ] 跳答指示是否一目了然？
- [ ] 答案记录方式是否统一？
- [ ] 页面布局是否便于阅读和作答？
- [ ] 字体大小是否合适？

### 说明与引导
- [ ] 是否有清晰的调查说明？
- [ ] 是否有保密性说明？
- [ ] 是否有答题指导？
- [ ] 重要术语是否有即时定义？

---

## 四、信度与效度检查

### 信度（一致性）
- [ ] 同样受访者重复测量是否会得到一致答案？
- [ ] 不同访员询问是否会得到一致答案？
- [ ] 问题措辞是否稳定（无歧义）？

### 效度（准确性）
- [ ] 答案是否真实反映了要测量的概念？
- [ ] 是否存在系统偏差？
- [ ] 问题是否与分析目标直接相关？
- [ ] 是否存在无关变量的干扰？

---

## 五、预调查评估准备

### 评估计划
- [ ] 预调查样本量是否足够（建议25-50份）？
- [ ] 预调查人群是否与目标人群相似？
- [ ] 评估方法是否确定（专题小组/深度访谈/田野预调查）？
- [ ] 行为编码方案是否准备就绪？
- [ ] 访员评估表是否准备就绪？

### 追问问题准备
- [ ] 是否准备了理解追问问题？
- [ ] 是否准备了叙述性答案追问？
- [ ] 是否准备了评估性问题？
