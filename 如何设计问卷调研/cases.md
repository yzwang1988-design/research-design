# 问卷设计案例库

## 一、人口统计学问题

### 年龄/出生日期

**❌ 问题设计（有缺陷）**
```
你多大了？
```
**问题**：
- 答案形式不明确（可能回答"中年""30多岁"等）
- 社会期望偏差（某些人群不愿透露年龄）

**✅ 优化设计**
```
1. 你上一次过生日时，几岁了？
   ____岁

或

2. 你的出生日期是？
   ____年____月____日
```
**改进**：
- 明确答案形式（数字）
- 降低敏感度（通过出生日期计算年龄）

---

### 收入

**❌ 问题设计（有缺陷）**
```
你的收入是多少？
```
**问题**：
- 术语未界定（月薪/年薪/家庭收入？）
- 敏感度高，缺乏缓冲
- 答案形式不明确

**✅ 优化设计**
```
问题前说明：
"收入信息有助于我们了解不同经济状况人群的观点。您的回答仅用于统计分析，绝对保密。"

问题：
去年一年，您以及所有与您住在一起的家庭成员，从工作和其他来源获得的总收入大约是？

□ 少于20,000元
□ 20,000-39,999元
□ 40,000-59,999元
□ 60,000-79,999元
□ 80,000-99,999元
□ 100,000元及以上
□ 拒绝回答
```
**改进**：
- 说明研究目的和数据用途
- 提供保密性保证
- 使用区间降低敏感度
- 提供"拒绝回答"选项

## 二、行为问题

### 医疗就诊

**❌ 问题设计（有缺陷）**
```
你最近看过病吗？
```
**问题**：
- "最近"时间框架模糊
- "看病"术语未界定（是否包括电话咨询？专科医生？）
- 答案形式不明确

**✅ 优化设计**
```
问题1：在过去14天里，您有几次亲自去看医生或与医生面对面讨论您的健康问题？
（不包括电话咨询，包括专科医生、精神科医生、中医等所有医疗专业人员）

____次

问题2：在同一时期（过去14天），您有几次通过电话与医生讨论健康问题？

____次
```
**改进**：
- 明确时间框架（14天）
- 界定"看病"范围（包括/排除）
- 分离不同行为（面对面vs电话）
- 明确答案形式（数字）

---

### 投票行为

**❌ 问题设计（有缺陷）**
```
去年十一月的选举中，你投票了吗？
□ 是
□ 否
```
**问题**：
- 社会期望偏差（不投票可能被视为"不良公民"）
- "否"答案可能受到负面评价

**✅ 优化设计**
```
问题前说明：
"我们知道，人们有时不去投票，是因为他们对选举不感兴趣，工作脱不开身，对候选人都不满意，或许多其他原因。"

问题：
请回想一下去年十一月的选举，您实际上有没有投票？
□ 投票了
□ 没有去投票
```
**改进**：
- 提供不投票的合理理由
- 平衡两种答案的社会可接受性
- 减弱"好公民"暗示

---

### 饮酒行为

**❌ 问题设计（有缺陷）**
```
你昨天一共喝了几杯酒？
____杯
```
**问题**：
- 社会期望偏差（如果昨天喝得比平时多）
- 缺乏参考基准
- 不知道"典型"行为

**✅ 优化设计**
```
问题1：当您喝酒时，通常喝几杯？
____杯

问题2：您认为您昨天喝酒比平常多，比平常少，还是与平常差不多？
□ 比平常多
□ 比平常少
□ 与平常差不多

问题3：请问昨天您一共喝了几杯酒？
____杯
```
**改进**：
- 先建立基准（通常喝多少）
- 提供相对参考（与平常比较）
- 更容易报告"异常"行为

---

### 运动锻炼

**❌ 问题设计（有缺陷）**
```
你经常锻炼吗？
□ 是
□ 否
```
**问题**：
- "经常"术语未界定
- "锻炼"概念模糊（散步算不算？）
- 二分法丢失信息

**✅ 优化设计**
```
问题说明：
"下面询问的是中等强度体育活动，即会让您呼吸和心跳比平时加快的活动，如快走、慢跑、游泳、骑自行车等。"

问题：在过去7天里，您有几天进行了至少30分钟的中等强度体育活动？
（不包括散步，但包括快走）

□ 0天
□ 1天
□ 2天
□ 3天
□ 4天
□ 5天
□ 6天
□ 7天
```
**改进**：
- 明确定义"中等强度"
- 明确包含/排除范围
- 提供具体测量标准（30分钟）
- 使用时间框架（7天）

## 三、主观评价问题

### 健康状况自评

**❌ 问题设计（有缺陷）**
```
你的健康状况如何？
□ 好
□ 不好
```
**问题**：
- 评价维度不明确（体质？疾病？生活方式？）
- 类别过少，信息量小
- 缺乏中间选项

**✅ 优化设计**
```
问题：总的来说，您如何评价您目前的健康状况？
（请考虑您的身体状况、疾病情况、体能水平等各个方面）

□ 极好
□ 很好
□ 好
□ 尚好
□ 差
```
**改进**：
- 提供评价维度提示
- 5级量表提供足够区分度
- 类别标签清晰

---

### 满意度评价

**❌ 问题设计（有缺陷）**
```
你对公司满意吗？
□ 满意
□ 不满意
```
**问题**：
- 评价对象不明确（整体？薪资？工作环境？）
- 二分法过于简化
- 缺乏中间立场

**✅ 优化设计**
```
问题1：考虑到所有因素，您对目前工作的整体满意度如何？

□ 非常满意
□ 满意
□ 既不满意也不不满意
□ 不满意
□ 非常不满意

问题2-6：请分别评价以下方面的满意度（使用相同5级量表）：
- 薪资福利
- 工作内容
- 工作环境
- 职业发展机会
- 管理方式
```
**改进**：
- 明确评价对象
- 提供中立选项
- 分解维度获取详细信息
- 统一量表便于比较

## 四、观点/态度问题

### 政策观点

**❌ 问题设计（有缺陷）**
```
你是否同意：枪支管制法是必要的？
□ 同意
□ 不同意
```
**问题**：
- "枪支管制法"含义模糊（可能指各种政策）
- 双重问题（必要性vs支持度）
- "同意-不同意"形式认知复杂

**✅ 优化设计**
```
问题1：请问您赞成还是反对关于对有暴力犯罪前科的人购买手枪进行背景调查的法律？
□ 强烈赞成
□ 赞成
□ 既不支持也不反对
□ 反对
□ 强烈反对
□ 不了解这个问题

问题2：请问您赞成还是反对关于禁止攻击性武器销售的法律？
（使用相同选项）
```
**改进**：
- 具体化政策内容
- 单一维度
- 直接评价（赞成-反对）而非间接（同意-不同意）
- 提供"不了解"选项

---

### 多维度观点

**❌ 问题设计（有缺陷）**
```
给政府官员写信没有什么用，因为他们经常对老百姓的问题并不是真的那么感兴趣。
□ 同意
□ 不同意
```
**问题**：
- 包含两个议题：写信的价值 + 官员的兴趣
- 受访者可能部分同意

**✅ 优化设计**
```
问题1：您认为给政府官员写信对解决老百姓的问题有多大作用？
□ 非常有用
□ 比较有用
□ 不太有用
□ 完全没用
□ 不知道

问题2：您认为政府官员对老百姓的问题有多大兴趣？
□ 非常感兴趣
□ 比较感兴趣
□ 不太感兴趣
□ 完全不感兴趣
□ 不知道
```
**改进**：
- 拆分为单一维度问题
- 分别测量两个概念
- 便于分析两者关系

## 五、知识测量问题

### 事实知识

**❌ 问题设计（有缺陷）**
```
谁是美国第十三任总统？
□ 米勒德·菲尔摩尔
□ 托马斯·杰弗逊
□ 理查德·尼克松
```
**问题**：
- 错误选项过于明显（杰弗逊、尼克松与菲尔摩尔时代差距大）
- 缺乏区分度

**✅ 优化设计**
```
谁是美国第十三任总统？
□ 米勒德·菲尔摩尔
□ 扎卡里·泰勒
□ 约翰·泰勒
□ 不知道
```
**改进**：
- 错误选项似是而非（同时代、相似名字）
- 提供"不知道"选项
- 更好的区分度

---

### 自我报告知识

**❌ 问题设计（有缺陷）**
```
你熟悉垃圾分类政策吗？
□ 是
□ 否
```
**问题**：
- "熟悉"标准不明确
- 二分法丢失程度信息

**✅ 优化设计**
```
关于您所在城市的垃圾分类政策，您认为自己了解多少？

□ 非常了解（能准确分类所有垃圾）
□ 比较了解（知道主要分类标准）
□ 了解一点（知道要分类，但不清楚具体标准）
□ 完全不了解
```
**改进**：
- 明确了解程度的定义
- 提供行为指标
- 4级量表捕捉程度差异

## 六、敏感问题

### 非法/敏感行为

**❌ 问题设计（有缺陷）**
```
你吸食过大麻吗？
□ 是
□ 否
```
**问题**：
- 高度敏感，社会期望偏差严重
- 缺乏隐私保护
- 时间框架不明确

**✅ 优化设计**
```
问题前说明：
"这部分问题涉及个人健康行为。您的回答完全保密，不会与您的身份信息关联。请诚实回答，您的回答对我们了解公共健康状况非常重要。"

问题1：在您的一生中，您是否曾经吸食过大麻？
□ 是 → 继续问题2
□ 否 → 跳过问题2-3

问题2：在过去30天里，您是否吸食过大麻？
□ 是
□ 否

问题3：在过去30天里，您有几天吸食过大麻？
____天

（自填式问卷或CAPI方式呈现，确保隐私）
```
**改进**：
- 强调保密性
- 分层询问降低敏感度
- 使用自填方式保护隐私
- 明确时间框架

## 七、问卷格式案例

### 访员管理问卷格式

**❌ 格式设计（有缺陷）**
```
A1. 你工作吗？有就去A2，没有问A1a
A1a. 你以前工作过吗？...
```
**问题**：
- 指令与问题混杂
- 跳答不清晰
- 缺乏一致性

**✅ 优化设计**
```
A1. 在[参考日期]时，您是否有一份带薪的工作？
    □ 有 ............................ 继续A2
    □ 没有 .......................... 问A1a

A1a. 在[参考日期]之前，您是否曾有过一份带薪的工作？
     □ 有
     □ 没有 ........................ 跳至A6

A2. [在参考日期或之前您的最后一份工作]，您的工作属于哪个行业？
    _____________________________

[访员注意：请逐字念出问题，不要解释或改写]
```
**改进**：
- 指令与问题明确区分
- 跳答路径清晰
- 访员提示完整

---

### 自填问卷格式

**❌ 格式设计（有缺陷）**
```
1.您吸烟吗？是 否
如果否跳至3
2.您一天吸几支？______
3.您喝酒吗？
```
**问题**：
- 跳答不够醒目
- 布局混乱
- 问题间距不足

**✅ 优化设计**
```
═══════════════════════════════════════

1. 您是否吸烟？
   □ 是 → 继续第2题
   □ 否 → 跳至第3题

───────────────────────────────────────

2. 您通常一天吸多少支烟？
   □ 少于5支
   □ 5-10支
   □ 11-20支
   □ 超过20支

───────────────────────────────────────

3. 您是否饮酒？
   □ 是 → 继续第4题
   □ 否 → 跳至第5题

═══════════════════════════════════════
```
**改进**：
- 视觉分隔清晰
- 跳答指示醒目
- 问题间距充足
- 格式统一

## 八、综合案例：完整问卷片段

### 主题：工作满意度调查

**问卷设计**

```
═══════════════════════════════════════
工作满意度调查问卷
═══════════════════════════════════════

说明：
本问卷旨在了解员工对工作各方面的看法。您的回答将严格保密，仅用于统计分析。请根据您的真实感受作答。

═══════════════════════════════════════

第一部分：基本信息

Q1. 您在目前公司工作了多少年？
    □ 少于1年
    □ 1-3年
    □ 3-5年
    □ 5-10年
    □ 超过10年

Q2. 您的职位类别是？
    □ 基层员工
    □ 专业技术人员
    □ 中层管理人员
    □ 高层管理人员

═══════════════════════════════════════

第二部分：工作满意度

请使用1-5分评价以下方面：
1分=非常不满意，2分=不满意，3分=一般，4分=满意，5分=非常满意

Q3. 薪资福利满意度
    □1 □2 □3 □4 □5

Q4. 工作内容满意度
    □1 □2 □3 □4 □5

Q5. 工作环境满意度
    □1 □2 □3 □4 □5

Q6. 职业发展机会满意度
    □1 □2 □3 □4 □5

Q7. 管理方式满意度
    □1 □2 □3 □4 □5

Q8. 整体工作满意度
    □1 □2 □3 □4 □5

═══════════════════════════════════════

第三部分：开放式问题

Q9. 您认为公司最需要改进的方面是什么？
    ___________________________________
    ___________________________________

Q10. 其他意见或建议：
    ___________________________________
    ___________________________________

═══════════════════════════════════════

感谢您的参与！
═══════════════════════════════════════
```

**设计要点分析**：
1. 清晰的说明和保密承诺
2. 逻辑分组（基本信息→具体维度→开放问题）
3. 统一的评价量表便于比较
4. 适量的开放式问题获取深度信息
5. 视觉分隔清晰
6. 预计完成时间：5-8分钟
